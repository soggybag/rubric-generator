# Rubric Generator

A simple rubric generator. 

 Goals: Create digital Rubric
  
- Easily create a rubric from JSON
- Easily score rubric
  - Displays the score in the page
  - Sends score in JSON
- Share rubrics online 
- Automatically score rubrics
